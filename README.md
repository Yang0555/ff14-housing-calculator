# FF14 房屋抽獎概率計算工具 (FF14 Housing Lottery Probability Calculator)

這是一個專為《最終幻想14》（FF14）玩家設計的房屋抽獎（Lottery）綜合概率計算器。
當您在一個或多個伺服器投標多個房屋時，此工具可以幫助您評估中獎機率。

## 主要功能

1. **截圖 OCR 辨識**：支援拖放投標試算表截圖，或直接使用 `Ctrl + V` 貼上截圖，系統會自動辨識房號指令及參與人數。
2. **單間房屋中獎率**：計算每間投標房屋的獨立中獎率 ($P_i = \text{我的抽數} / \text{總抽數}$)。
3. **綜合累計中獎率**：使用 **泊松二項式分布 (Poisson Binomial Distribution)** 的動態規劃演算法，精確計算出「至少獲得 $K$ 間房產」與「剛好獲得 $K$ 間房產」的機率。
4. **即時編輯與微調**：辨識完成後，可在表格內直接修改或手動新增/刪除投標數據。
5. **程式診斷驗證**：內建自我測試，一鍵驗證數學邏輯和文字解析引擎是否百分之百準確。

## 輸入格式說明

工具主要辨識 FF14 常見的投標指令格式：
`/li <房屋地名> <區號> <房號> <總投標數> <我的投標數>`

### 範例：
- `/li Mist 18 42 2`
  - 表示投標 Mist (海霧村) 18 區 42 號。
  - 該房總投標數為 2（包含自己），我的投標數為 1 (預設值)。
  - 中獎率： $1 / 2 = 50.00\%$
- `/li Lavender 20 48 3 2`
  - 表示投標 Lavender Beds (薰衣草苗圃) 20 區 48 號。
  - 該房總投標數為 3，我的投標數為 2（例如與部屬或親友合作投 2 票）。
  - 中獎率： $2 / 3 = 66.67\%$
- `/li Lavender 14 49 4`
  - 表示投標 Lavender Beds (薰衣草苗圃) 14 區 49 號。
  - 該房總投標數為 4，我的投標數預設為 1。
  - 中獎率： $1 / 4 = 25.00\%$

上述三間房子的**綜合累計中獎機率**為：
- **獲得至少一間房** 的機率： **$87.50\%$**
- **剛好獲得一間房** 的機率： **$41.67\%$**
- **剛好獲得兩間房** 的機率： **$37.50\%$**
- **同時獲得三間房** 的機率： **$8.33\%$**

## 數學公式與原理

### 1. 單個房產中獎機率 $P_i$
若第 $i$ 間房產的總抽獎人數為 $N_i$，我的投標數為 $M_i$，則：
$$P_i = \min\left(1.0, \frac{M_i}{N_i}\right)$$

### 2. 獲得剛好 $k$ 間房的機率 $DP[i][j]$
使用動態規劃演算法遞迴更新：
- 設 $dp[j]$ 為在前 $i$ 間房屋中剛好中獎 $j$ 間的概率。
- 轉移方程：
  $$dp[j] = dp[j] \times (1 - P_i) + dp[j-1] \times P_i$$

### 3. 獲得至少 $k$ 間房的機率 $AtLeast[k]$
$$AtLeast[k] = \sum_{j=k}^{n} dp[j]$$

這與 FF14 實際抽獎情況完全契合，幫助玩家最大化購置個人房或部屬房的配置策略！
